Title and Purpose:
scheduling_app
A scheduling app GUI that tracks users, customers, and appointments.

Author:
Amanda Lixey alixey1@wgu.edu

App Version: 1.0
Date: 6/21/2023

Software Information:
IDE Version: IntelliJ IDEA Community 2023.1
Java SDK Version: Java 17.0.6
JavAFX version: JavaFX -20
MySQL Connector Driver Version: mysql-connector-j-8.0.33

Directions:
1. Launch the application
2. Login with a valid username and password
3. Select the page you would like to visit. To view the contact schedule, click view contact schedules. To view, add, or update an appointment, click the view appointments button.
To view, add, or update a customer, click the view customers button.
4. Logout and exit when you are done.

Additional Report:
On the transition screen, the user can see their next upcoming appointment. If they do not have an upcoming appointment, it will say no upcoming appointments.
